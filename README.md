# SPACEAPPS_NASA_HACKHATHON

## Instalaciones previas:

- Java mayor a 17 
- Maven

## Cómo ejecutar:

- Windows:
```
mvnw.cmd spring-boot:run
```

## Linux:

* Dar permisos de ejecución
```
chmod +x mvnw
```
* Ejecutar
```
./mvnw spring-boot:run
```

## Ver servidor:
Poner en el buscador:
```
http://localhost:8080/
```
